# Data Management API

## Project Overview

This project is a RESTful Data Management API built using **Flask** and **MySQL**. It provides efficient backend operations for managing database records through well-structured API endpoints. The project demonstrates CRUD functionality and API testing with Postman for reliable backend data management.

---

## Features

- Developed a RESTful API using Flask for backend data management.
- Supports Create, Read, Update, and Delete (CRUD) operations.
- Uses MySQL for persistent data storage.
- Tested and validated 10+ API endpoints using Postman.
- Ensures seamless synchronization between API requests and the MySQL database.

---

## Tech Stack

- **Language:** Python
- **Framework:** Flask
- **Database:** MySQL
- **API Testing:** Postman

---

## Usage

- Start the Flask server.
- Use Postman or any API client to send requests to the available endpoints.
- Perform CRUD operations on the MySQL database.
- Verify the database updates in real time.
